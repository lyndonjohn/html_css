HTML & CSS Exercise

Table

Instructions:
  1. Create a Social Media Table with the following headers:
    • No.
    • Social Media
    • Description
    • URL
  2. List all social media sites you can think of.
  3. In Description column, just generate a lorem dummy text.
  4. In URL column, make an href tag that links to their website.
    • Make sure to include their full website, for example: https://www.website.com
    • When you click the link, open the website in new tab.
    • Use #6495ed for link color.
  5. Header should have a background of #d3d3d3 and text color is #000000.
  6. Border color should be #000000.
  7. Collapse table border and set the padding to 1em.
  8. Save your table with socmed.html filename.

Form

Instructions:
  1. Create a registration form with the following fields:
    • First Name
    • Last Name
    • Email
    • Birthday
    • Gender
      • Use radio button input type.
    • Civil Status (Single, Married, Separated, Widowed)
      • Use select input type.
    • Select Social Media (Facebook, Instagram, Twitter)
      • Use checkbox input type.
    • Password
  2. Set the proper input type for each fields.
  3. Add Submit and Cancel buttons at the bottom.
    • Submit and Cancel should have a different style.
  4. Design your own form. Be creative.
